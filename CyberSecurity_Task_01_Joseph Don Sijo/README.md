# Cyber Security Task 01: Personal Security Audit & Cyber Awareness Assessment

## Intern: Joseph Don Sijo
## 🎯 Objective
The purpose of this task is to understand basic cybersecurity concepts by assessing my own digital security practices and identifying potential risks. This task emphasizes the core principle that cybersecurity starts with securing yourself before securing systems and networks.

## 📁 Folder Contents

*   **`Device_Security_Assessment`** - Documentation and screenshots verifying my operating system version, update status, antivirus installation, and firewall configuration.
*   **`Password_Policy`** - An analysis of strong password principles, the risks of password reuse, an explanation of Multi-Factor Authentication (MFA), and a customized 5-point personal password policy.
*   **`Account_Security_Review`** - A review of three major online accounts checking for MFA, strong passwords, and recovery email configurations.
*   **`Threat_Research`** - Definitions, real-world examples, and prevention methods for common threats: Phishing, Malware, Ransomware, Social Engineering, and Data Breaches.
*   **`Awareness_Poster`** - A simple visual awareness poster designed to promote safe digital habits.
*   **`README.md`** - This index file summarizing the task contents.

## 🧠 Key Takeaways
Completing this task highlighted that cybersecurity is not only about tools and hacking. It fundamentally begins with understanding personal risks, developing secure daily habits, and protecting information responsibly.

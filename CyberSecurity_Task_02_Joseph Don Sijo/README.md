# Cyber Security Task 02: CIA Triad & Risk Assessment
## Intern: Joseph Don Sijo
## 🎯 Objective
The purpose of this task is to understand the core principles of Cyber Security and learn how organizations identify and manage security risks. The foundation of this exercise is the CIA Triad (Confidentiality, Integrity, Availability), which underpins all cybersecurity practices.

## 📁 Folder Contents

*   **`CIA_Triad_Research.md`** - An explanation and comparison of Confidentiality, Integrity, and Availability, including real-world examples.
*   **`Cyber_Attack_Case_Study.md`** - An analysis of a major real-world cyber attack, evaluating how it impacted the CIA Triad and how it could have been prevented.
*   **`Risk_Assessment_Table.md`** - A simulated risk assessment for a college, identifying 5 key assets, potential threats, impact levels, and overall risk scores.
*   **`Threat_Analysis.md`** - Research detailing common security threats such as Insider Threats, Malware, Phishing, Weak Passwords, and Unpatched Systems, along with prevention methods.
*   **`Security_Recommendations.md`** - 10 actionable security recommendations based on the risk assessment.
*   **`README.md`** - This index file summarizing the task contents.

## 🧠 Key Takeaways
This task emphasized that before learning advanced security tools and ethical hacking techniques, a cybersecurity professional must first understand what they are protecting, why it needs protection, and what specific risks exist. By conducting a simulated risk assessment and acting as a security consultant, I gained practical experience in threat modeling and strategic security planning.

# Cyber Security Task 02: CIA Triad & Risk Assessment

## 🎯 Objective
The purpose of this task is to understand the core principles of Cyber Security and learn how organizations identify and manage security risks[cite: 5]. The foundation of this exercise is the CIA Triad (Confidentiality, Integrity, Availability), which underpins all cybersecurity practices[cite: 5].

## 🧠 Key Takeaways
This task emphasized that before learning advanced security tools and ethical hacking techniques, a cybersecurity professional must first understand what they are protecting, why it needs protection, and what specific risks exist[cite: 5]. By conducting a simulated risk assessment and acting as a security consultant, I gained practical experience in threat modeling and strategic security planning[cite: 5].

# Ethical Hacking Task 02: Network Scanning & Service Enumeration
## Intern: Joseph Don Sijo
## 🎯 Objective
The purpose of this task is to understand the second phase of Ethical Hacking: Scanning and Enumeration. This exercise focuses on safely identifying active hosts, open ports, running services, and basic network information using authorized methods on local or permitted systems.

## 📁 Folder Contents

*   **`Nmap_Scan_Report.pdf`** - A comprehensive professional scan report detailing the scan date, target system, commands used, running services, OS detection results, observations, and security recommendations.
*   **`Screenshots/`** - Visual documentation verifying the Nmap installation (`nmap --version`), local machine scans (`nmap localhost`), service version detection (`-sV`), and OS detection (`-O`).
*   **`README.md`** - This index file summarizing the task contents.

## 🛠️ Tools Used
*   **Nmap (Network Mapper):** Utilized for network discovery, port scanning, service version detection, and operating system fingerprinting. 

## 🧠 Key Takeaways
Completing this task provided hands-on experience with discovering a system's attack surface. By analyzing which ports are open and determining if those services are strictly necessary, I gained a deeper understanding of how misconfigured services can become security risks and the importance of minimizing an environment's exposure.

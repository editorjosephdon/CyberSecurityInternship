# Ethical Hacking Task 01: Information Gathering & Reconnaissance
## Intern: Joseph Don Sijo
## 🎯 Objective
The purpose of this task is to understand the first phase of Ethical Hacking Reconnaissance: Information Gathering. This involves performing passive reconnaissance using publicly available sources to collect information about a target domain without attempting to access or exploit any systems.

## 📁 Folder Contents

*   **`Reconnaissance_Report.pdf`** - A comprehensive report detailing the target information, WHOIS results, DNS enumeration (A, MX, NS, TXT records), identified website technologies, security headers, and `robots.txt`/sitemap findings.
*   **`Screenshots/`** - Visual evidence documenting the WHOIS lookup, DNS queries, and file analysis.
*   **`Research_Notes.md`** - Notes gathered during the investigation of the target website's HTTP security headers and overall footprint.
*   **`README.md`** - This index file summarizing the task contents[cite: 6].

## 🧠 Key Takeaways
This task demonstrated that a successful ethical hacker spends significant time understanding a target before testing it. By analyzing DNS records, server headers, and exposed files like `robots.txt` and `sitemap.xml`, I learned how to build a detailed technological footprint of an organization using purely passive techniques. Developing these strong reconnaissance skills is essential for conducting more effective and responsible security assessments.
